About the Project

Our project is a web-based messaging application that allows users to communicate through private and group chats. This is designed for students, teams, and small groups who need a simple and organized way to stay connected with eachother. The end goal is to build a platform that is secure, and user-freindly. We will apply Scrum principles, Github collaboration, and modern software engineering workflows.
